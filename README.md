## 利用の流れ
1.『未完星コレクション』のリンクをクリックする

2.ログイン/新規登録ページに遷移<br>
    ユーザーネーム・メールアドレス・パスワードを入力。

3.メイン画面である、自分の宇宙が表示されるページに遷移<br>
    メイン画面でクリックした場所によって、次のいずれかの工程に進む。

4.『原石登録』をクリックする<br>
    『未完成』のアイデアや作品を、アイデアの原石として登録する。<br>
    具体的には、アイデアの名称・詳細情報を入力する。その後、タグ(勉強・イラスト等)を選択し、原石がどんなものであるかを決定する。<br>
    また、公開設定をすることができる。公開しない設定をした場合は、原石が自分にしか表示されず、未来の自分が実現するためのものになる。公開する設定をした場合は、他のユーザーにも原石が表示されるようになり、誰かにアイデアの原石を託すことができる。この場合は自分ではない誰かにアイデアの原石を実現してもらう。<br>
    最後に登録ボタンをクリックすることで、アイデアの原石を登録することができる。

5.『コレクション』をクリックする<br>
    登録した原石を一覧で見ることができる。原石をクリックすると、原石の詳細と完了ボタンが表示される。アイデアの原石を実現できた場合、完了ボタンをクリックする。<br>
    すると、アイデアの原石が星座に変化する。星座も、タブを切り替えることで一覧で見ることができる。

6.『宇宙を見る』をクリックする<br>
    ログインした際と同じ、メイン画面に遷移する。<br>

以上の手順で、『未完成』のアイデアを原石として登録し、実現できたら星座として完成させる。この一連の流れで利用する。

### 開発における工夫した点
アイデア出しではメンバー全員でブレインストーミングを行い、様々なアイデアを出すように工夫しました。その際には曖昧なイメージや単語などに関わらず、ひたすら思いついたものを書いていき、そこから大事にしたいポイントを考えていきました。最終的に、『あ"きら"めるをあ"きら"めないに』というテーマに決定しました。<br>
また、実装フローの決定にMVP（Minimum Viable Product）を意識するように工夫しました。今回制作した『未完星コレクション』に必須な機能は何かを洗い出し、優先順位をつけて実装していきました。さらに、実装する際にも各担当（バックエンド・フロントエンド・インフラ）で連携をとりつつ、進捗報告を丁寧に行いながら開発を進めていきました。<br>
