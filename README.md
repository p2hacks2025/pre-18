## 利用の流れ
1.『未完星プロジェクト』のリンクをクリックする

2.ログイン/新規登録ページに遷移
    ユーザーネーム・メールアドレス・パスワードを入力。

3.メイン画面である、自分の宇宙が表示されるページに遷移
    メイン画面でクリックした場所によって、次のいずれかの工程に進む。

4.『原石登録』をクリックする
    『未完成』のアイデアや作品を、アイデアの原石として登録する。
    具体的には、アイデアの名称・詳細情報を入力する。その後、タグ(アイデア・イラスト等)を選択し、原石がどんなものであるかを決定する。
    また、星・星座のどちらかの形態を選ぶ。これは、アイデアの原石を誰に実現してほしいのかを示している。
    星を選択した場合は未来の自分。時間ができたり、技術が向上した、未来の自分が実現する。
    星座を選択した場合は、他のユーザー。自分ではない誰かに、アイデアの原石を託して実現してもらう。
    最後に登録ボタンをクリックすることで、アイデアの原石が登録できる。

5.『コレクション』をクリックする
    登録した原石を一覧で見ることができる。原石をクリックすると、原石の詳細と完了ボタンが表示される。アイデアの原石を実現できた場合、完了ボタンをクリックする。
    すると、アイデアの原石が星・星座に変化する。星・星座も、タブを切り替えることで一覧で見ることができる。

6.『宇宙を見る』をクリックする
    ユーザーが今まで完成させた星・星座を見ることができる。星・星座はタグで管理されており、対応したタグがついているロゴをクリックすることで、星・星座を見ることができる。

以上の手順で、『未完成』のアイデアを原石として登録し、実現できたら星・星座として完成させる。この一連の流れで利用する。
